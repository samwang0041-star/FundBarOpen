# FundBar

把基金和股票放进 macOS 菜单栏。  
抬眼就能看到今天的涨跌、估值和当日盈亏，不用反复拿手机，也不用切到行情 App。

[下载最新版本](../../releases/latest)

## 为什么会想用它

你未必需要一个复杂行情终端。很多时候，你只是想在工作时快速看一眼：今天涨了还是跌了，手里大概赚了还是亏了。

FundBar 把这层最常看的信息，直接放进菜单栏。

## 你能得到什么

- 菜单栏直接显示主资产涨跌幅
- 基金和 A 股一起看，最多支持 5 个自选
- 可填写持仓，直接看到当日盈亏估算
- 基金盘中看估算，官方净值发布后自动切换
- iCloud 同步、开机启动、应用内更新提示
- 没有广告，没有会员墙

## 适合谁

- 平时主要盯着 Mac 工作
- 不想反复拿手机看行情
- 只关心少量自选和今天的变化
- 想要一个轻量工具，而不是重型行情 App

如果你需要港股、美股、分时图、交易下单或复杂分析，这个版本还不适合。

## 安装

1. 打开 [Releases](../../releases/latest) 下载最新 DMG
2. 将 `FundBar.app` 拖到 `Applications`
3. 首次启动后，它会出现在 macOS 菜单栏

## 当前支持

- 基金：6 位基金代码
- 股票：A 股 6 位代码
- 股票输入支持 `600519`，也支持 `sh600519` / `sz000858`
- macOS 14.0 及以上

## 常见问题

**Q: 支持哪些股票？**  
A: 当前支持沪深 A 股。输入 6 位代码即可，也兼容带 `sh` / `sz` 前缀的写法。

**Q: 支持哪些基金？**  
A: 当前支持 6 位基金代码的开放式基金。

**Q: 数据多久刷新一次？**  
A: 交易时段内会自动刷新，也可以手动点击“立即刷新”。

**Q: 为什么基金显示的是“本地估算”？**  
A: 因为基金盘中没有官方实时净值。FundBar 会基于最近披露持仓和市场行情做参考估算，方便你快速判断方向和大致盈亏。

**Q: 换了 Mac 后数据还在吗？**  
A: 如果两台设备登录同一个 iCloud 账号，且系统允许同步，自选列表会自动同步。

**Q: 安全吗？**  
A: 它不会要求注册账号，没有广告、没有会员墙，也不会因为不打赏而限制功能。
