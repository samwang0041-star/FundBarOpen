# 📈 FundBar — 股票/基金 盯盘小工具

> 搬砖摸鱼的时候，一眼就知道今天情况。

FundBar 是一个 macOS 菜单栏小工具，不用打开任何 App，就能随时瞄一眼自选基金和股票的实时涨跌。

---

## ✨ 功能特色

| 功能 | 说明 |
|------|------|
| 📊 **菜单栏实时显示** | 主显示资产的涨跌幅直接出现在菜单栏，随时一眼就看到 |
| 📈 **支持基金和股票** | 基金、股票混合管理，最多自选 5 个资产 |
| 🏷️ **主显示切换** | 选一只最关心的设成主显示，菜单栏只看它 |
| 🧮 **盘前本地估算** | 开盘前根据最近持仓自动估算涨跌，不用干等 |
| 📋 **收盘看官方净值** | 基金净值发布后自动切换到官方数据 |
| 💰 **盈亏估算** | 输入持有份额，自动计算当天估算盈亏金额 |
| ☁️ **iCloud 同步** | 多台 Mac 之间自动同步自选列表，换电脑无缝衔接 |
| 🔄 **自动更新检查** | 启动时自动检查新版本，有更新会在面板内提示 |
| 🌙 **开机自启** | 支持开机启动，打开电脑就能用 |

---

## 📸 界面预览

打开菜单栏面板后，你会看到：

- **顶部** — FundBar 标题 + 当前市场状态（开盘中 / 午休 / 已收盘）
- **主显示卡片** — 涨跌幅大字显示 + 估值/净值 + 盈亏金额
- **自选列表** — 所有自选资产的实时涨跌一览
- **底部控制栏** — 添加资产、手动刷新、开机启动开关

## 📥 安装方法

### 下载安装

1. 前往 [📦 Releases 页面](../../releases/latest) 下载最新的 `FundBar_v1.0.dmg`
2. 双击打开 DMG 文件
3. 将 **FundBar** 拖入 **Applications** 文件夹
4. 在 Applications 中双击打开 FundBar

> **⚠️ 首次打开提示**
>
> 因为没有上架 App Store，macOS 可能会提示「无法打开」或「已损坏」。
>
> **解决方法：** 打开终端（Terminal），输入以下命令：
>
> ```bash
> sudo xattr -cr /Applications/FundBar.app
> ```
>
> 输入密码后回车，然后再双击打开 FundBar 即可。

## 🛠️ 使用说明

### 添加自选资产

1. 点击菜单栏中的 FundBar 图标，打开面板
2. 点击「**添加资产**」按钮
3. 输入**基金代码**（如 `001437`）或**股票代码**（如 `sh600519`）
4. 选择资产类型（基金 / 股票）
5. 可选：输入持有份额，用于计算盈亏
6. 可选：勾选「设为主显示」

### 管理自选

- 点击「**管理**」按钮进入管理模式
- 可以**编辑**、**删除**资产，或**切换主显示**
- 点击「**完成**」退出管理模式

### 数据来源说明

| 状态 | 说明 |
|------|------|
| **实时** | 股票实时行情数据 |
| **本地估算** | 基金盘中按持仓自动估算 |
| **盘前估算** | 开盘前的预估数据 |
| **官方净值** | 基金收盘后的官方净值（净值发布后自动切换） |

## 💡 为什么写这个软件

我是一个在银行上班的个人开发者。

平时工作的时候，经常会想看一眼自己买的基金和股票今天涨了还是跌了。但每次都要打开手机、解锁、找到 App、等它加载……这个过程太打断工作了。

我就想：能不能有个东西，直接在 Mac 的菜单栏里显示涨跌幅，不用动手机，不用切窗口，眼睛瞄一下就知道今天行情怎么样？

找了一圈，没有特别满意的，于是就自己写了一个。

最开始只是给自己用的小工具，后来越做越完善——加了基金估算、盈亏计算、iCloud 同步、多资产管理。慢慢觉得，可能也有人和我一样需要这样一个轻量的盯盘工具，就把它整理出来分享给大家了。

**这个工具没有任何广告、没有会员、没有后台偷偷跑数据。** 它就是一个纯粹的、干净的小工具。

## ❓ 常见问题

**Q: 支持哪些基金？**
A: 支持天天基金网收录的所有开放式基金。输入 6 位基金代码即可。

**Q: 支持哪些股票？**
A: 支持沪深两市（A 股）所有股票。代码格式：`sh` + 6 位代码（上证）或 `sz` + 6 位代码（深证）。

**Q: 数据多久刷新一次？**
A: 开盘期间自动定时刷新，也可以手动点击「立即刷新」。

**Q: 为什么基金显示「本地估算」？**
A: 基金盘中没有实时净值，FundBar 会根据最近披露的持仓数据自动估算涨跌。收盘后基金公司发布官方净值时会自动切换。

**Q: 换了 Mac 数据还在吗？**
A: 如果两台 Mac 登录了同一个 iCloud 账号，自选列表会自动同步。

**Q: 打开提示「已损坏」怎么办？**
A: 打开终端，输入 `sudo xattr -cr /Applications/FundBar.app`，回车输入密码即可。

**Q: 最低系统要求？**
A: macOS 14 (Sonoma) 及以上。
